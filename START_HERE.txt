QUICK START (3 COMMANDS):

1. Build smart contract:
   cd contracts
   cargo build --target wasm32-unknown-unknown --release

2. Deploy to testnet (requires stellar-cli and account):
   stellar contract deploy --network testnet --source-account YOUR_ACCOUNT \
     --wasm target/wasm32-unknown-unknown/release/stellar_micro_insurance.wasm

3. Start frontend (requires Node.js):
   cd frontend
   npm install
   npm start
